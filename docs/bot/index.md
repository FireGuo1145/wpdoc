# 风梨小助手

风梨小助手是由成员FireGuo主要维护和开发的多功能小助手，采用ts+js的高性能koishi框架。

功能概览：

当前可用的互动功能：

  /cave  回声洞

  /propose  向群友求婚

  /waifu  娶群友

  /lh  离婚

当前可用的工具：

  /be  查询基岩版服务器信息

  /hitokoto  随机一言

  /je  查询Java版服务器信息

  /mudl  音乐下载

  /news  每日60s新闻

  /gs帮助  原神查询菜单

  /sr帮助  崩铁查询菜单

  /ww帮助  鸣潮查询菜单

当前可用的娱乐功能：

  /bj  21点游戏

  /chat  AI聊天（15积分）

  /checkin  签到获取积分

  /ddz  斗地主

  /gn  猜数字

  /homo  恶臭数字论证器

  /jf  查看积分

  /jrrp  今日人品

  /zz  积分转账

可在侧边栏查看详细功能使用帮助
